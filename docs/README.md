## 三、当前最重要的项目判断

如果你是新对话里的助手，先记住这几点：

1. 当前主开发页面是 **Amazon运营推进器**
2. 当前真正重点是：
   - 首页工作台
   - 模块3运营数据主卡区
3. 当前优先级是：
   - **先保功能，再精修样式**
4. 当前默认不要主动做的事：
   - 不要主动扩数据库
   - 不要主动改后端接口契约
5. 当前已有一部分“按运营查看本人产品”的逻辑，但它仍然是前端模拟，不是后端真实权限
